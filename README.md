# Student Database Management System

## 📌 Description

A command-line application to manage student records using Python and SQLite.

## 🚀 Features

* Add Student
* View Students
* Update Student
* Delete Student

## 🛠 Tech Stack

* Python
* SQLite

## 📊 How It Works

1. User selects option from menu
2. Program performs database operation
3. Data is stored using SQLite

## ▶️ How to Run

```bash
python main.py
```
